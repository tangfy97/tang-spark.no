Echoknot for iOS

Support and guide for version 1.1

This page is aligned with the shipped iOS codebase (`CFBundleShortVersionString 1.1`). It covers the live feature set, troubleshooting, and data behavior.

Contact Support Back to Product Intro

Current latest build: 1.1

At a glance

  • Capture knots with word, translation, usage, memory anchor, and tags.
  • Generate echo stories from selected knots (minimum 3, maximum 8).
  • Save recent echoes and favorite echoes (up to 5 favorites).
  • Export knots and echoes as JSON or PDF from Settings and Dashboard.
  • Theme and language controls: Light, Dark, System; EN, zh-CN, zh-TW, System.

Features in v1.1

Context-first knot capture

Save the word you heard, your translation, optional usage sentence, memory anchor, and tags for recall.

Echo chamber workflow

Select knots in one language, generate a short story plus translation, and keep recent echoes for reuse.

Favorite echoes

Mark saved echoes as favorites for fast revisit. The current cap is 5 favorites.

Export and reset tools

Export data as structured JSON or readable PDF. You can also delete all data and reset to first-run state.

Display and language settings

Set app language, text size, learning level (A2, B1-B2, C1), and appearance mode.

Local-first storage

Data is persisted locally on-device with a primary snapshot, backup snapshot, and defaults fallback.

Quick Start

  1. Open Echoknot and choose your learning language and level.
  2. Tap the + action to capture your first knot.
  3. Add meaning, usage, and a memory anchor tied to a real moment.
  4. Open Echo chamber and select at least 3 knots from one language.
  5. Create an echo story and favorite the best echoes for repeat review.

Support / Help Center

How do I edit or delete a knot?

Open a knot from the dashboard list, then use the edit action in knot details. Delete is available in knot details as a destructive action.

Why can't I generate an echo story?

Echo generation is disabled when fewer than 3 knots are selected, when selected knots are from mixed languages, when quota is exhausted, or when network/provider access fails.

How do favorites work?

You can favorite saved echoes. Favorites are capped at 5. Unfavorite any item to free space.

Where can I export JSON or PDF?

Use Settings > Data export buttons, or open the Dashboard export menu in the all-knots area. Both JSON and PDF formats are supported.

What does reset remove?

Reset deletes knots, recent echoes, favorite echoes, and preferences, then restores first-run defaults.

Where do I change theme and language?

In Settings, open Appearance to set theme (Light, Dark, System), app language, and text size.

Does Echoknot sync across devices?

Not in v1.1. The app is local-first in this build. Cloud sync/sign-in is planned as a future optional capability.

What data is sent when I use AI features?

Only the text needed for the action you trigger is sent to the configured AI provider. There are no background AI calls by default.

Privacy & Data Handling

  • No account is required in the current build.
  • Knots, settings, and echoes are stored locally on-device by default.
  • AI actions are user-triggered only and send minimal required text.
  • Export is available as JSON or PDF for backup/migration.
  • Always-on cloud sync is not enabled in v1.1.

Troubleshooting

AI unavailable

  • Confirm internet access, then retry.
  • Check if AI quota has been reached.
  • Try a smaller selection and regenerate.

Export failed

  • Retry export in the same format.
  • Confirm the iOS share sheet appears correctly.
  • Reopen the app and try again.

Echo button disabled

  • Select at least 3 knots.
  • Select knots from one language only.
  • Verify quota is not paused.

Unexpected reset after reinstall

  • Data is local to the device and app installation.
  • Use JSON/PDF export regularly if you need backups.
